## Development

Run the dev server from the **`site/`** directory (not the project root):

```bash
cd site
npm run dev
```

If images or content look stale (e.g. broken hero photos), stop old servers and restart fresh:

```bash
cd site
npm run dev:clean
```

Then open the URL printed in the terminal and hard-refresh the browser tab.

Manage the background server with `astro dev stop`, `astro dev status`, and `astro dev logs`.
